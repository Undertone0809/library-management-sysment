# 图书管理系统

这是一个简易的图书管理系统

## 特性

- 查询图书信息
- 添加图书信息
- 修改图书信息
- 删除图书信息
- 图书数据持久化保存

## 快速上手

```shell
git clone https://github.com/Undertone0809/library-management-sysment
```

- 导包：打开终端命令行，输入下面的命令

```shell
pip install -r requirements.txt
```

- 运行`init_data.py`初始化数据
- 在第三方包下载完之后，直接运行`app.py`就好了
